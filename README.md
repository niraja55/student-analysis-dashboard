# student-analysis-dashboard
Interactive Student Performance Analysis dashboard built using Power BI. Power BI project for analyzing and visualizing student performance.
🎓 Student Performance Analysis Dashboard Using Power BI

📖 Project Description

Student Performance Analysis is a Business Intelligence project developed using Microsoft Power BI to analyze and visualize student academic performance through interactive dashboards. The project transforms raw educational data into meaningful insights, helping identify trends, compare subject-wise performance, monitor attendance, and evaluate overall student achievement.

The dashboard features interactive charts, KPIs, slicers, and filters that allow users to explore data from multiple perspectives. It supports data-driven decision-making for educators and administrators by providing clear and actionable insights into student performance.

🛠️ Tech Stack

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel (Dataset)
- Data Modeling
- Data Visualization

✨ Key Features

- Interactive and dynamic dashboard
- Student performance overview with KPIs
- Subject-wise performance analysis
- Attendance analysis
- Performance trend visualization
- Filters and slicers for better data exploration
- Clean, responsive, and user-friendly interface
- Actionable insights for academic decision-making
